### How to Create Your First Note

1. Launch Soul.md AI
2. Click **"New Note"**
3. Title your note (e.g., "Welcome to Soul.md")
4. Start writing in Markdown or WYSIWYG mode
5. Type `[[` to link to another note
6. Type `#` to add a tag
7. Press `Ctrl + S` to save
8. Note appears in sidebar

### How to Use AI Writing Assistant

1. Open a note
2. Highlight text or place cursor where you want help
3. Press `Ctrl + Shift + A` or click the AI icon
4. Choose action:
   - **Autocomplete** — Finish your sentence
   - **Summarize** — Create summary of selected text
   - **Expand** — Elaborate on short text
   - **Rewrite** — Change tone
   - **Fix Grammar** — Correct errors
   - **Translate** — Change language
5. AI generates response
6. Accept, edit, or regenerate

### How to Create Bidirectional Links

1. In any note, type `[[`
2. Start typing existing note name
3. Select note from dropdown or type new name
4. Finalize with `]]`
5. Example: `[[Quantum Mechanics]]`
6. The linked note gets a backlink panel showing connections
7. Click backlinks to navigate

### How to Use Semantic Search

1. Click **"Search"** tab (or press `Ctrl + Shift + F`)
2. Type a natural language query, for example:
   - `"ideas for my novel about time travel"`
   - `"research on renewable energy"`
   - `"recipes I saved"`
   - `"meeting notes from last quarter"`
3. Search understands meaning (not just keywords)
4. Results ordered by relevance
5. Use filters for tags, folders, or date ranges
6. Click result to open note

### How to Navigate the Knowledge Graph

1. Click **"Graph"** tab
2. See all notes as nodes, links as edges
3. **Zoom** — Mouse wheel
4. **Pan** — Drag background
5. **Drag node** — Manual repositioning
6. **Click node** — Open note
7. **Double-click node** — Focus on local graph
8. **Color settings** — Color by folder, tag, or date
9. **Export** — Save graph as image

### How to Use Auto-Tagging

1. Open any note
2. Click **"Tags"** panel
3. Click **"Suggest Tags"** button
4. AI analyzes note content
5. Suggested tags appear (e.g., `#science`, `#research`, `#physics`)
6. Click `+` to accept any tag
7. Click **"Apply All"** to add all suggested tags
8. Tags automatically appear in tag cloud

### How to Import from Obsidian or Notion

**From Obsidian:**
1. Click **"File"** → **"Import"** → **"Obsidian"**
2. Select your Obsidian vault folder
3. Choose what to import (notes, folders, tags, links)
4. Click **"Import"**
5. All notes appear in Soul.md

**From Notion:**
1. Export your Notion workspace as HTML
2. Click **"File"** → **"Import"** → **"Notion"**
3. Select the exported .zip file
4. Click **"Import"**
5. Notes converted to Markdown

### How to Enable Daily Notes

1. Click **"Settings"** → **"Core Plugins"**
2. Enable **"Daily Notes"**
3. Set new note location (e.g., "Journal/")
4. Set template (or use default)
5. Click **"Open Today's Note"** from sidebar
6. Daily note opens automatically
7. Templates can include `{{date}}`, `{{time}}`, `{{title}}`
